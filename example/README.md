# Business Card Scanner Example

This is an example Flutter application that demonstrates how to use the `business_card_scanner` package.

## Features

- Scan business cards using the device camera
- Import business cards from the device gallery
- View extracted contact information
- View raw OCR text

## Getting Started

1. Clone the repository
2. Run `flutter pub get` to install dependencies
3. Run the app on your device or emulator

## Usage

1. Tap the camera icon to take a photo of a business card
2. Or tap the gallery icon to select a business card from your device
3. View the extracted contact information
